# coelacanth logo

- `logo-transparent.png` is the primary transparent PNG, preserving the original canvas.
- `logo-transparent-cropped.png` is the transparent PNG trimmed for placement with padding.

Design direction:

- Zig-like warm yellow/orange palette.
- Angular heavy-metal wordmark energy.
- Lowercase `coelacanth` wordmark.
- Subtle coelacanth/living-fossil symbolism in the first `c`.
- Small terminal/contract-frame notches to echo typed NDJSON frames and stdout/stderr separation.
